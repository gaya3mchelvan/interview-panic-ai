# Interview Panic AI 🎤

Interview Panic AI is a mock interview preparation assistant designed to help students practice technical interviews and improve confidence.

---

## Problem Statement

Many students face:
- interview anxiety
- lack of preparation
- communication fear

This project helps users practice role-based interview questions in a simple interactive way.

---

## Features

- Role-based interview preparation
- Technical interview questions
- Beginner-friendly interaction
- Career preparation support

---

## Supported Roles

- Software Engineer
- Data Analyst
- Machine Learning Engineer

---

## Workflow

User selects target role  
↓  
System generates interview questions  
↓  
User practices responses  
↓  
Improved interview confidence
